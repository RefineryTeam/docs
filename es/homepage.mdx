## ¿Qué es Refinery?

Refinery es un equipo de desarrollo enfocado en crear software de alta calidad para comunidades de jugadores y desarrolladores.

Nuestro trabajo incluye:

- **Plugins de Minecraft** desarrollados para Paper, Spigot y software de servidor compatible.
- **Bots de Discord** con comandos slash modernos, automatización e integraciones.
- **Sistemas e integraciones de servidor** adaptados a comunidades específicas.
- **Bibliotecas y recursos de código abierto** para desarrolladores.
- **Servicios de desarrollo a medida** para comunidades que requieren funcionalidades únicas.

Ya sea que gestiones un pequeño servidor comunitario o una gran red, Refinery se propone ofrecer software confiable, configurable y fácil de administrar.

## Acerca de esta wiki

La wiki contiene la documentación de todos los proyectos públicos de Refinery.

La documentación suele incluir:

- Guías de instalación
- Referencias de configuración
- Documentación de comandos
- Nodos de permisos
- Soporte de placeholders
- Documentación de API (cuando esté disponible)
- Preguntas frecuentes
- Guías de resolución de problemas
- Guías de migración entre versiones

A medida que se publiquen nuevos proyectos, su documentación se añadirá aquí.
